BrainLab – Deep-learning, Pairwise Image Registration

GitHub - Coming soon!
At BrainLab, I am developing an unsupervised, deep-learning method for deformable, pairwise image registration, with a focus on brain tissue.

This deep-learning registration is formulated as a function that maps an input image pair to a deformation field that aligns these images.

The aim is to integrate this method into BrainLab's MIRACL pipeline, complimenting non-ML based registration methods such as ANTs, which are effective, but slower by several orders of magnitude.

Reincarnating Bob Ross – Character-Level LSTM on YouTube Transcripts

GitHub
The goal of this project is to make the following as straight-forward as possible: Perform LSTM character-level generation on automatically extracted YouTube transcripts.

Personally, I tackled this project to re-create the dear, late Bob Ross' painting videos.

Prompt: “welcome back. certainly glad you could join us today, it's a fantastic day here. let's have them run all”

Generated: ... “the colors across the screen, there we go. now then, we'll take a little bit of the bright-red and let's put some of the same colors and then we'll put a little bit of the bright-red in the sky. there we go. now, little touch of the bristles, we'll go right into the yellow-ochre”
